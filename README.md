# Flowmen Project

This repository contains the Flowmen project. Below are the steps to set up and run the project using GitHub Codespaces.

## Getting Started

### Prerequisites

- A GitHub account
- Access to the `sltpl-repositories/Flowmen` repository

### Steps to Start the Project Using GitHub Codespaces

1. **Open GitHub and Sign In:**
   - Go to [GitHub](https://github.com) and sign in with your credentials.

2. **Navigate to the Repository:**
   - Find the repository named `sltpl-repositories/Flowmen`.
   - You can use the search bar at the top of the page or navigate through your repositories.

3. **Open the Repository:**
   - Click on the repository name to open it.
   - Click on the branch option and select the branch named `common`.

4. **Start a Codespace:**
   - In the repository, click on the green `<> Code` button near the top right of the page.
   - In the dropdown, there will be an option `Codespaces`. Click on `Create codespace on common`.

5. **Wait for the Codespace to Initialize:**
   - GitHub will start setting up your codespace. This process might take a few minutes as it installs the necessary dependencies and sets up the development environment.

6. **Use the Codespace:**
   - Once the codespace is ready, it will open in a new tab, showing the Visual Studio Code interface.

### Set Up and Start the Frontend

1. Open a terminal in the codespace (usually found in the menu under `Terminal > New Terminal`).

2. Navigate to the `Frontend` folder:
   ```bash
   cd Frontend
3. Install dependencies:
    ```bash
    npm install
4. Start the frontend:
    ```bash
    npm run dev
5. `ctrl`+ click on the link available

### Set Up and Start the Backend
1. Open a new terminal.

2. Navigate to the BackEnd folder:
    ```bash
    cd BackEnd

3. Install dependencies:
    ```bash
    npm i

4. Start the backend:
    ```bash
    npm run start
